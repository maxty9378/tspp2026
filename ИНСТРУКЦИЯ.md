# Инструкция по использованию рейтинга в PuzzleBot

## Быстрый старт

1. **Установите зависимости:**
   ```bash
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. **Запустите скрипт:**
   ```bash
   python get_rating.py
   ```

3. **Получите HTML файлы:**
   - `rating.html` - полная версия с красивым дизайном
   - `rating_puzzlebot.html` - упрощённая версия для PuzzleBot

## Использование в PuzzleBot

1. Откройте файл `rating_puzzlebot.html`
2. Скопируйте всё содержимое файла
3. В PuzzleBot добавьте блок "HTML" или "Веб-страница"
4. Вставьте скопированный HTML код

## Автоматическое обновление

Для автоматического обновления рейтинга:
- Настройте периодический запуск скрипта (например, через cron или планировщик задач Windows)
- Или используйте веб-хук в PuzzleBot для запуска скрипта при необходимости

## Настройка

Все параметры находятся в начале файла `get_rating.py`:
- `NOCODB_URL` - адрес вашего NocoDB
- `API_TOKEN` - ваш API ключ
- `BASE_ID` - ID базы данных
- `TABLE_NAME` - название таблицы (по умолчанию "День 1")
